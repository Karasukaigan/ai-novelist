# 粗略看了一下，感觉就是给function calling包装了个标准接口？

mcp，感觉就是给function calling包装了一个标准接口
然后所有遵守该接口的客户端，能够兼容所有遵守此接口的服务端
不像传统function calling，只能靠代码移植
mcp可以即插即用
本质上，对于ai模型本身来说，并没有区别，都是工具调用


## 试着实现一下，langchain好像就是支持mcp的客户端，无需手动写适配。

四行核心代码：

from langchain_mcp_adapters.client import MultiServerMCPClient

client = MultiServerMCPClient()

tools = await client.get_tools()

await client.close() ———— 短期感觉用不上，我并不需要重启啥的
